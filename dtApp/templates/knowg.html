Welcome to the Knowledge graph page.

Knowledge graph for the three-storey structure

In this page, the bill of materials (BoM) is imported and displayed. On this page the Bill of Materials is displayed. The definitions for the Bill of Materials are given in the Excel file BoM.xlsx which can be found in the `static' folder of the App. To make changes to the Bill of Materials you should edit the Excel file BoM.xlsx directly.

To visualise the knowledge graph you will need to have neo4j installed and running on your local machine and then follow this link Neo4j browser interface page.
